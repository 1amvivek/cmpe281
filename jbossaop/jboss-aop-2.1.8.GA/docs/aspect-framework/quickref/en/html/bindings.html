2.3. Bindings

XML 

            
   <bind {name} pointcut {cflow} >
      [ <interceptor-ref name/> |
        <stack-ref name/> |
        <advice aspect name/> |
        <before aspect name/> |
        <after aspect name/> |
        <around aspect name/> |
        <throwing aspect name/> |
        <finally aspect name/>  ]*
   </bind>

Annotation

Usage: class (if annotated @InterceptorDef), method (if class annotated @Aspect)
